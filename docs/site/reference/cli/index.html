CLI reference

The cgc command-line interface provides a suite of tools for indexing, querying, and managing your code graph.

Core commands

CommandUsageDescriptionOptions
index cgc index [PATH] Index a repository or directory. --dependency
--deep
--force
watch cgc watch [PATH] Monitor a directory for changes and update the graph in real-time.
stats cgc stats [--all] Display statistics about the current indexed repository. --all

Querying the graph

CommandUsageDescriptionOptions
query cgc query [TYPE] [TARGET] Execute relationship-based queries.
Types: callers, callees, hierarchy, dead-code
find cgc find [QUERY] Search for code symbols (functions, classes, etc.) by name or content. --fuzzy
visualize cgc visualize [--query Q] Generate a visual graph of the codebase or a specific query. --query

Bundle management

CommandUsageDescriptionOptions
bundle create cgc bundle create --name NAME Export the current index to a .cgc bundle file. --name
bundle load cgc bundle load [PATH|NAME] Load a local bundle or download one from the registry.
bundle search cgc bundle search [QUERY] Search the CGC bundle registry.

System & configuration

CommandUsageDescriptionOptions
config cgc config [SUBCOMMAND] Manage global configuration settings.
Subcommands: set-db, list
doctor cgc doctor Run diagnostics to verify installation and backend connectivity.
mcp cgc mcp Start the Model Context Protocol (MCP) server. Typically invoked by AI assistants.
Global Options
--database Override the default database for a single command.
--verbose Enable detailed logging for debugging.
--version Display the version of CodeGraphContext.
