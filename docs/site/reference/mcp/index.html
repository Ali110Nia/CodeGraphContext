MCP tool reference

When CodeGraphContext is running as an MCP server, it exposes a set of tools that AI agents can use to explore and understand your codebase.

Code search & discovery

CommandDescriptionOptions
find_code Find relevant code snippets related to a keyword or phrase. query, fuzzy_search, repo_path
list_indexed_repositories List all repositories that are currently available in the graph database.

Relationship analysis

CommandDescriptionOptions
analyze_code_relationships The primary tool for deep architectural analysis. query_type, target, repo_path
calculate_cyclomatic_complexity Get the complexity score of a specific function. function_name, path
find_most_complex_functions Identify technical debt by listing the most complex functions in the codebase. limit

Graph management

CommandDescriptionOptions
add_code_to_graph Instruct the agent to index a new directory or file. path, is_dependency
watch_directory Start monitoring a directory for real-time updates. path
switch_context Switch the server's focus to a different indexed project. context_path

Advanced querying

CommandDescriptionOptions
execute_cypher_query A "power user" tool that allows the AI to run raw Cypher queries against the graph. cypher_query
visualize_graph_query Generates a visualization URL for a specific Cypher query. cypher_query

Bundle operations

CommandDescriptionOptions
load_bundle Load a portable .cgc bundle into the session. bundle_name
search_registry_bundles Search for available bundles in the remote registry. query
